# Ads Art

An extension that replaces ad slots with art from galleries.

## Privacy

All processing happens locally on the user's machine. The extension analyzes page content and replaces detected ad slots in the browser, and it does not send user data to a server for processing.

## Credits

Artworks come from the open-access programs of the [Art Institute of Chicago](https://www.artic.edu/open-access) and [The Metropolitan Museum of Art](https://www.metmuseum.org/art/collection/search-open-access). Metadata and images stay with their respective institutions.
